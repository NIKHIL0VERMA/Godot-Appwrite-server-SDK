# Appwrite SDK for headless Godot

This sdk provide support for Auth, Databases, Storage, Functions, Teams, and more.

[SDK Usage Example](https://nikhil0verma.github.io/Godot-Appwrite-server-SDK/)

> [!IMPORTANT]
> This project is community-maintained and is not officially maintained by Appwrite.

## Installation

1. Open the AssetLib tab inside the Godot 4 editor.
2. Search for Appwrite server SDK.
3. Download and install plugin.
4. Enable the plugin from Project -> Project Settings -> Plugins.

## Getting Started

The plugin automatically intializes itself using values from .env. You can also set them at runtime using the `set_endpoint()`, `set_project()` and `set_self_signed()` methods.

### Quick Test connection to Appwrite server 

```gdscript
extends Node

func _ready():
    Appwrite.ping()
```
